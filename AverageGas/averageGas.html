What is the purpose of a function?

The reason that we use functions is to be able to repeat the same actions without redundancy. For example, if we need to average out the gas prices for one individual and then average them for another individual, instead of having to write code to average each person's gasoline costs we can simply creat calGasAvg, as we have done here, and run both individuals through that function. This saves us time and makes our code less complex and thereby, less error-prone.
